Welcome to Twilio-fueled Telephone Pictionary!

In Telephone Pictionary, each person starts by writing down a word or phrase. That word or phrase gets passed to another person who reads it, and then draws a picture representing that topic. That drawing then gets passed to a new person, who looks at the picture and writes a sentence describing it. The game continues that way, with players alternating between drawing or writing, until each player has interacted with every starting topic.

General Gameplay

  1. The host will create a game and send the game id to other players.
  2. Players can join by texting JOIN and the game id.
  3. The host can see who joins a game and decide when to start it. Once the host starts a game, everyone will be prompted to send in an initial word or phrase.
  4. After everyone has sent their initial word/phrase, each participant will be sent one of those phrases, draw it, and send back the drawing.
      How to draw and send images:
    • Draw in a Notes or Google Keep mobile application, screenshot the image, and send it.
    • Draw in the web browser using something like Sketch.io, screenshot it, and send the image.
    • Draw the image with a pen and paper, take a picture, and send the picture.
  5. Next, each player will receive one of those images, and should write a sentence to describe the image.
  6. The game continues until each player has written or drawn every starting phrase. Then, the game ends and players can see a gallery of the game.

Important commands:

  • Text CREATE to create a new game (You can only be in one game at a time.).
  • Text START to begin the game once everyone has joined. If you are the host, you will receive a notification each time a player joins the game. Only the host can start a game.
  • Text LEAVE to leave a game. This will end the game for everyone.
  • Text JOIN and the game id to join an existing game. A game cannot be joined after it is started by the host. You can only be in one game at a time. Example: JOIN abcd
  • Text STATUS and the game id to see the status of an existing game. Example: STATUS abcd
  • Text REPEAT PROMPT if you are in a game to receive your prompt again.
